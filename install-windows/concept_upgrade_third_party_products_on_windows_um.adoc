= サードパーティ製品のアップグレード
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
JREなどのサードパーティ製品がWindowsシステムにインストールされている場合は、Unified Managerでそれらの製品をアップグレードできます。

これらのサードパーティ製品を開発している企業は、定期的にセキュリティの脆弱性を報告しています。このソフトウェアの新しいバージョンへのアップグレードは、ご自身のスケジュールで行うことができます。
