= Snapshotイベント
:allow-uri-read: 
:icons: font
:imagesdir: ../media/

Snapshotイベントは、Snapshotのステータス情報を提供します。これにより、Snapshotの潜在的な問題を監視できます。影響範囲別にイベントがグループ化され、イベント名、トラップ名、影響レベル、ソースタイプ、および重大度が表示されます。
