= NVRAMバッテリイベント
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NVRAMバッテリイベントは、バッテリのステータス情報を提供します。これにより、潜在的な問題を監視できます。影響範囲別にイベントがグループ化され、イベントとトラップの名前、影響レベル、ソースタイプ、および重大度が表示されます。
