= ストレージシェルフイベント
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ストレージシェルフイベントは、ストレージシェルフに異常がある場合に通知します。これにより、潜在的な問題を監視できます。影響範囲別にイベントがグループ化され、イベントとトラップの名前、影響レベル、ソースタイプ、および重大度が表示されます。
