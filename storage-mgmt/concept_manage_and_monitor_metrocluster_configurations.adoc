= MetroCluster設定の管理と監視
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Unified Manager Web UIでMetroCluster構成を監視することで、MetroCluster over FCおよびIP構成に接続の問題がないかどうかを確認できます。接続の問題を早期に検出することで、MetroCluster構成を効率的に管理できます。
