= クォータの管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ユーザクォータとグループクォータを使用すると、ユーザまたはユーザグループが使用できるディスクスペースの量やファイル数を制限できます。ディスクとファイルの使用量、ディスクに設定されているさまざまな制限など、ユーザクォータとユーザグループクォータの情報を表示できます。
