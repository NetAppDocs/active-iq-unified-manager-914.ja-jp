= Storage VMレポートのカスタマイズ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Storage VMレポートを作成して、ボリューム情報を分析したり、全体の健全性とストレージの可用性を確認したりできます。たとえば、レポートを作成して、ボリューム数が上限に達しているSVMを表示したり、停止したSVMを分析したりできます。
