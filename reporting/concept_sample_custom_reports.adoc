= サンプルのカスタムレポート
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以下に記載するサンプルのカスタムレポートを使用して、潜在的な問題を未然に特定して対処することができます。

このセクションのレポートのリストはすべてを網羅しているわけではなく、時間の経過とともに増えていきます。ドキュメントに関するフィードバックを提供することで、このセクションに追加するカスタムレポートを提案できます。

[NOTE]
====
レポートを管理するには、アプリケーション管理者またはストレージ管理者のロールが必要です。

====
